# ExScroller Game Studio 2 - Vision

**Version**: 0.2.0
**Date**: 2026.02.07

## 1. ExScrollerとは

> **紙という物理メディアでしか成立しないゲーム体験を創造するプラットフォーム**

サーマルプリンタを「ゲームコンソール」として再定義し、デジタルゲームにはない独自の体験を提供する。

### 1.1 紙メディアの本質的特性

| 特性 | 説明 | ゲームへの応用 |
|------|------|----------------|
| **永続性** | プレイログが物質として残る | 証拠、記念、コレクション |
| **一方向性** | 基本は巻き戻せない | 緊張感、不可逆な選択 |
| **分断性** | 切れる、渡せる、重ねられる | 情報共有、カードゲーム |
| **書込み可能** | ペンで追記できる | 入力デバイス化 |
| **有限性** | 紙・電力は消耗品 | リソース管理ゲーム |

### 1.2 デジタルゲームとの差別化

```
デジタルゲーム          ExScroller
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
画面は消える        →   紙は残る
無限にリトライ      →   一発勝負の緊張感
孤独なプレイ        →   紙を渡す物理的交流
データは見えない    →   履歴が可視化される
どこでも同じ        →   その場・その時だけ
```

## 2. Studio 2 の目的

### 2.1 既存Studio (v1) との違い

| 項目 | Studio v1 | Studio 2 |
|------|-----------|----------|
| **エディタ** | コードのみ | ビジュアル + コード |
| **プレビュー** | 印刷時のみ | リアルタイム |
| **オブジェクト操作** | 座標手入力 | ドラッグ/回転/リサイズ |
| **シーン遷移** | コードで記述 | ノードグラフ |
| **ゲームパターン** | 手動実装 | テンプレート選択 |

### 2.2 ターゲットユーザー

1. **ノンプログラマー**: ビジュアルエディタでゲーム作成
2. **プログラマー**: コードエディタで高度なカスタマイズ
3. **両方**: ビジュアルで配置 → コードで微調整

## 3. 入力デバイス仕様

### 3.1 ボタン入力 (v10.7.0対応)

| ボタン | ビットマスク | 用途例 |
|--------|-------------|--------|
| A | 0x01 | 決定、進行 |
| B | 0x02 | キャンセル、戻る |
| X | 0x04 | サブアクション |
| Y | 0x08 | メニュー |
| L | 0x10 | ページ戻り |
| R | 0x20 | ページ進み |

```javascript
// SDK使用例
section.jumpIfBtn(BTN.A, 'next');           // A → next
section.jumpIfBtn(BTN.A | BTN.B, 'action'); // A or B → action
section.waitButton(BTN.ANY);                // 任意ボタン待ち
```

### 3.2 フェーダー入力 (v10.7.0対応)

10kΩポテンショメーターによるアナログ入力（0-4095、12bit ADC）

| モード | 説明 |
|--------|------|
| AUTO | 自動紙送り（デフォルト） |
| BUTTON | ボタンで紙送り制御 |
| FADER | フェーダー値で紙送り速度/方向制御 |

```javascript
// フェーダーによる分岐
section.setFeedMode(FEED_MODE.FADER);
section.jumpIfFader(PGP.OP.GE, 2048, 'high');  // 50%以上 → high
section.jumpIfFader(PGP.OP.LT, 2048, 'low');   // 50%未満 → low
section.waitFader(PGP.OP.EQ, 0);               // 最小まで待機
```

### 3.3 ゲームパターン別入力マッピング

| パターン | 主入力 | 補助入力 | フェーダー用途 |
|----------|--------|----------|----------------|
| REVEAL | A (進行) | B (戻る) | - |
| LOOP | A (進行) | B (巻き戻し) | 巻き戻し速度 |
| FEEDER | - | - | 紙送り速度 |
| SYNC | A/B (選択) | X/Y (アクション) | HP/リソース表示 |
| BACKLOG | A (継承) | B (上書き) | 復活コスト |
| GENERATOR | A (生成) | L/R (ページ) | ランダム度 |
